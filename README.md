# Складський облік — PHP/MariaDB MVC
Веб-додаток для кількісного обліку руху матеріалів на складах з автоматичним розрахунком списання за витратою ресурсів.

--------------------------------------------------------------------------------------------------------------------------
## Журнал змін (останні зверху)

### Мобільна адаптація та покращення UI (травень 2026)
- **Додано кнопку-гамбургер** для відкриття сайдбару на мобільних пристроях
- **Autocomplete на мобільних** – замінено на нативний select для уникнення проблем з клавіатурою
- **Головний діапазон дат** – кнопки пресетів (Поточний місяць, Минулий місяць, Сьогодні, Поточний рік) тепер у сітці 2×2 однакової ширини
- **Мета-інформація в модалках** – автор та дата останньої зміни/створення відображаються дрібним шрифтом в одному рядку з кнопками
- **Виправлено групування в звітах** – рекурсивне розгортання/згортання через класи та data-атрибути

### Звіт по використанню ресурсів (травень 2026)
- **Новий звіт** – деталізація витрати ресурсів по матеріалах та складах
- **Групування** трирівневе: Матеріал → Склад → Дні (з можливістю розгортання/згортання)
- **Колонки звіту**: 
  - Лічильник (показник одометра/таймера)
  - Δ ресурсу (витрата ресурсу за період)
  - Норма списання
  - Поправка (у %)
  - Вхідне сальдо, Прихід, Витрата, Вихідне сальдо
- **Фільтри**: тип ресурсу (мотогодини, пробіг тощо), період
- **Множинний вибір складів** – всі склади, що мають норми для вибраного типу ресурсу
- **Експорт в Excel** – два листи:
  - `Деталізація` – ієрархічна структура як на екрані
  - `Для зведеної` – плоска таблиця для зведених таблиць Excel
- **Друк** – оптимізований для паперового формату
- **Кнопки** "Все розгорнути" / "Все згорнути" для зручної навігації
- **Додаткове відображення годин** – для типів ресурсів з форматом `dec2` та одиницею `год` можна ввімкнути показ витрати також у годинах:хвилинах (галочка "Додатково в годинах")

### Уніфікація форм та звітів (травень 2026)
- **Всі довідники** (склади, матеріали) приведено до однакового стилю:
  - Однакова структура таблиць
  - Однакові кнопки дій (редагувати, видалити)
  - Однакове відображення порожнього стану (empty-state)
- **Всі звіти** (по складу, по матеріалу, по ресурсу) мають єдиний стиль:
  - Однакові фільтри та кнопки
  - Однакове групування зі стрілочками
  - Однакові підсумкові рядки та картка-футер
- **Всі модальні вікна** мають єдине оформлення:
  - Однакові заголовки, кнопки, відступи
  - Мета-інформація (автор, дата) внизу в одному рядку з кнопками
- **Сторінка "Типи ресурсів"** доповнена колонкою "В годинах" та галочкою "Додатково в годинах"

### Додано мета-інформацію в таблиці
- Таблиця `resource_rates` доповнена полями: `author`, `created_at`, `updated_at`
- Таблиця `resource_types` доповнена полем `show_hours` (галочка "Додатково в годинах")

### Виправлення логіки розрахунку балансу
- Виправлено врахування рознесених по днях списань (`spread_by_day`)
- Баланс розраховується накопичувально на основі всіх рухів (надходження + списання)
- Ручні списання виділяються помаранчевим кольором з позначкою ⚠️
